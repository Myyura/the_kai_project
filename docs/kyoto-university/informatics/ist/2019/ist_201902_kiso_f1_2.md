# 京都大学 情報学研究科 知能情報学専攻 2019年2月実施 専門科目 F1-2

## **Author**
祭音Myyura (co-authored with GPT 5.2 extended thinking)

## Description (English)

Give context-free grammars that generate the following languages. In all parts, the alphabet (terminal character set) is $\{0, 1\}$.

Q.1 $\{ w \mid w \text{ contains at least three 1s} \}$

Q.2 $\{ w \mid \text{the length of } w \text{ is odd and its middle symbol is a } 0 \}$

Q.3 $\{ w \mid w = w^R, \text{ where } \mathcal{R} \text{ denotes writing backwards, that is, } w \text{ is a palindrome} \}$

## **Kai**
### Q.1

* Terminals: $\Sigma={0,1}$
* Nonterminals: $\{S,A\}$
* Start symbol: $S$
* Productions:

$$
  \begin{aligned}
  S &\to A \ 1 \ A \ 1 \ A \ 1 \ A \\
  A &\to 0A \mid 1A \mid \varepsilon
  \end{aligned}
$$

Here (A) generates any binary string, i.e. (L(A)=\Sigma^*).

### Q.2

A string has odd length and middle symbol $0$ iff it can be written as

$$
w = x\ 0\ y \quad\text{with } |x|=|y|.
$$

### Grammar (G_2)

* Terminals: $\Sigma=\{0,1\}$
* Nonterminals: $\{S\}$
* Start symbol: $S$
* Productions:

$$
  S \to 0 \mid 0S0 \mid 0S1 \mid 1S0 \mid 1S1
$$

Intuition: Start with the center $0$. Each step adds **one symbol on the left and one on the right**, preserving odd length and keeping the center fixed.

### Q.3

* Terminals: $\Sigma=\{0,1\}$
* Nonterminals: $\{S\}$
* Start symbol: $S$
* Productions:

$$
  S \to \varepsilon \mid 0 \mid 1 \mid 0S0 \mid 1S1
$$
