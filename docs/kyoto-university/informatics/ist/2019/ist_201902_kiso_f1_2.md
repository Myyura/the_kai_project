# 京都大学 情報学研究科 知能情報学専攻 2019年2月実施 専門科目 F1-2

## **Author**
[itsuitsuki](https://github.com/itsuitsuki)

## Description (English)

Give context-free grammars that generate the following languages. In all parts, the alphabet (terminal character set) is $\{0, 1\}$.

### Q.1

$\{ w \mid w \text{ contains at least three 1s} \}$

### Q.2

$\{ w \mid \text{the length of } w \text{ is odd and its middle symbol is a } 0 \}$

### Q.3

$\{ w \mid w = w^R, \text{ where } \mathcal{R} \text{ denotes writing backwards, that is, } w \text{ is a palindrome} \}$
