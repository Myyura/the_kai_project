# 京都大学 情報学研究科 知能情報学専攻 2025年8月実施 情報学基礎 F1-1

## **Author**
[itsuitsuki](https://github.com/itsuitsuki)

## **Description (English)**
### Q.1
Suppose that a sequence $\{x_k \mid k \text{ is a positive integer}\}$ satisfies the recurrence formula,
$$
x_1 = 1, x_2 = 2, x_3 = 1, x_m = a x_{m-1} - a x_{m-2} + b x_{m-3}
$$
where $a$ and $b$ are real numbers, and $m$ is an integer greater than or equal to 4. Let $\boldsymbol{y}_m = (x_m \ x_{m-1} \ x_{m-2})^{\mathrm{T}}$. Answer the following questions, where $n$ is an integer greater than or equal to 4. $\mathrm{I}$ is the identity matrix of order 3, and $A^{\mathrm{T}}$ stands for the transpose of a matrix $A$.

(1) Find a matrix $A$ which satisfies $\boldsymbol{y}_{m+1} = A \boldsymbol{y}_m$.

(2) Find $a$ and $b$ for which $A - \mathrm{I}$ is invertible. For the values $a$ and $b$, determine $(A - \mathrm{I})^{-1}$.

(3) Let $a = -\frac{1}{2}$ and $b = \frac{1}{4}$. Compute the eigenvalues of $A$, and $\lim_{n \to \infty} A^n$.

(4) With the same condition as (3), express $\sum_{k=1}^n x_k$ using $\sum_{m=4}^n \boldsymbol{y}_m$, and compute $\lim_{n \to \infty} \sum_{k=1}^n x_k$ using the results of (1) – (3).

### Q.2

In the following, we assume that all elements of matrices are real numbers. In what follows, a matrix with $m$ rows and $n$ columns is called an $m \times n$ matrix. Let $\mathrm{O}_{m \times n}$ be the zero matrix of $m$ rows and $n$ columns, and $\mathrm{I}_n$ be the identity matrix of order $n$. The rank of a matrix $M$ is denoted by $\mathrm{rank}(M)$. Let $A$ be a $k \times m$ matrix, and $B$ be an $m \times n$ matrix. Prove each of the following statements (1) to (5) if it holds, or if it does not hold, give a counterexample of a pair $(A, B)$ and disprove the statement with it.

(1) $\mathrm{rank}(AB) \leq \mathrm{rank}(B)$.

(2) If $k = m$ and $A$ is non-singular, $\mathrm{rank}(AB) = \mathrm{rank}(B)$.

(3) $\mathrm{rank}(A) + \mathrm{rank}(B) - m \leq \mathrm{rank}(AB)$.

(4) If $AB = \mathrm{O}_{k \times n}$, $\mathrm{rank}(A) + \mathrm{rank}(B) \leq m$.

(5) If $n = k$ and $AB = \mathrm{I}_k$, $\mathrm{rank}(A) = k$.
